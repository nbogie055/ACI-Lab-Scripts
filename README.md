# ACI-Lab-Scripts

-Install python 2.7 or higher - https://www.python.org/downloads/

-Install cobra sdk  

    navigate to https://<APIC address>/cobra/_downloads/

    Download the two whl files

    pip install \*directory\path*\requirements.txt

    python .\scriptname
